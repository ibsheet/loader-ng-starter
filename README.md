# loader-ng-starter

[`@ibsheet/loader`](https://github.com/ibsheet/loader) starter for angular

## clone project

```bash
git clone https://github.com/ibsheet/loader-ng-starter ibl-ng-starter
cd ibl-ng-starter
```

## setup and start

#### Using npm:

```
npm install
npm start
```

#### Using yarn:

```
yarn
yarn start
```
